# Pizza Sales Analysis Dashboard | Power BI

## Project Overview
This project presents an interactive Power BI dashboard developed to analyze pizza sales performance, revenue trends, customer preferences, and key business KPIs.

## Key Features
- Revenue Analysis
- Order Trend Analysis
- Best & Worst Selling Pizzas
- Category-wise Performance
- KPI Tracking Dashboard

## Tools Used
- Power BI
- DAX
- Power Query
- Microsoft Excel

## Business Insights
- Identified top-performing pizza categories.
- Analyzed peak sales periods.
- Evaluated customer ordering patterns.
- Tracked revenue and order performance through KPIs.
